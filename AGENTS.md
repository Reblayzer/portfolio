# Agent instructions for this repo

## Populating real content

**If the user is asking you to fill the site with their real bio, projects,
experience, or résumé, read `docs/POPULATING-CONTENT.md` before doing
anything else.** It explains what's already built, what's a placeholder,
where each kind of content lives, and house rules (no em dashes, tone,
flagship vs card decisions, etc.).
